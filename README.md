# api-call (postman)

# Short Project Description:
This project demonstrates how to test and manage a RESTful API using Postman. 

# Setup Instructions:
- [Install Laravel project and set up your local server (e.g., php artisan serve).]
- [Make sure your database is configured in the .env file.]
- [Run migrations using: php artisan migrate.]
- [Start the Laravel server: php artisan serve.]
- [Open Postman and create a new request.]
- [Use the correct endpoints, for example:]
- [GET /api/students – fetch all students]
- [POST /api/students – create a new student]
- [GET /api/students/{id} – fetch a specific student]
- [PUT /api/students/{id} – update a student]
- [DELETE /api/students/{id} – delete a student]
- [Set header: Content-Type: application/json for POST/PUT requests.]
- [Click Send to view the response from the API.]
